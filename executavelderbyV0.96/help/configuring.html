The Oracle Execution Configuration

In the main manu bar, select Setup at the Test Cases option.
The oracle execution may be configured:
  • To limitate the number of instants to be analyzed. It may be interesting because:
    • The analysis will stop when the limit is reached
    • The user may not want to wait for all the simulation instants
  • To limitate the number of identified failures. It may be interesting because:
    • The analysis will stop when the limit is reached
    • The user may not want to wait for all failure identifications
    • The model may present a huge number of failures and a list with thousands of failures may not be useful
  • To select an analysis hypothesis
