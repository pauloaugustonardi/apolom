Executing the Oracle

To execute the oracle, the user must:
  • Have the specification already created
  • Have mapped the specification properties to the respective signal (line) models
  • Open the instrumented model on Simulink (or other supported tool, in the future), which is in the current project directory
  • Simulate the instrumented model, via Simulink
  • Configurate the oracle
  • Execute the model by selecting execute at Test Cases option, in the main menu bar
