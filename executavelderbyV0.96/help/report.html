The Report

The report is composed by a main table and a sequence table
The report presents all the instants where a failure occur and/or if a spurious value is detected
The main table have 6 columns:
  • The OIU name, i.e., the description of a oiu that has at least one disrespected rule
  • The instant when the OIU was disrespected
  • If there is a failure w.r.t. the main rule
  • If there is a failure w.r.t. at least one constraint rule
  • If there is a failure w.r.t. at least one saveguard rule
  • If there is a spurious value detected
Clicking on one of the main table lines, the sequence table will show:
  • The next 100 values from all the signals involved on the failure
Each oracle execution generates a log of the test case. The user can navigate through the results by clicking on the previous and next buttons, or by typing the test case number on the text box and clicking on the jump button
