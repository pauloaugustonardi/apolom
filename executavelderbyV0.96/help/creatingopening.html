Creating and Opening An Oracle Project

To be able to generate an automated oracle, the user must first create or open a new oracle project.

Creating a new Oracle Project

To open a new oracle project:
  1. Go to Project at the main menu bar (Figure 1)
  2. Select New
  3. Type a name to the project (Figure 2)
  4. Select a path to save the project
  5. Select the Simulink-like model, if there is one at this point
    1. If there is no model at this point, you can select one later, clicking on Project->OpenModel
    2. You will need a model before executing the oracle. But you can create the specification without it
  6. Select the model type: Simulink or Scicos (just Simulink is implemented by now)
  7. Type a project description, if desirable
  8. Press the Create button
  • Back to the previous page
  • Return to the main page

Opening a pre-existing Oracle Project

To open a new oracle project:
  1. Go to Project at the main menu bar
  2. Select Open
  3. Select the directory where your previously created project is
  4. Press the Ok button
  • Back to the previous page
  • Return to the main page
