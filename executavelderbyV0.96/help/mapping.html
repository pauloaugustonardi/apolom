Mapping the properties to their respective lines

A property represents a line on the Simulink-like model.
A property contains a value for each simulation instant.
Examples of property are: pressure, temperature, altitude, velocity, etc...

How to define a property

 To define a property:
  1. Go to the instrumentation tab (Figure 3)
  2. On the upper-left green box, enter the property name at its respective text box and press Add
  3. A new property will appear at the property list on the left
  • Back to the previous page
  • Return to the main page

How to map a property to its respective line

 To map a property to its respective line:
  1. Go to the instrumentation tab (Figure 3)
  2. Select a property clicking on its respective name at the relation list, on the upper-center pink box
  3. Select a line at the Model Signals list on the upper-right blue box OR select a line directly from the model at the center of the screen
  4. Press the Insert button on the upper-right blue box
  • Back to the previous page
  • Return to the main page
